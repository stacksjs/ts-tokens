# Candy Machine UI

A React + Vite app demonstrating Candy Machine creation and minting with **ts-tokens**.

## Setup

```bash
bun install
bun dev
```

## Features

- Connect Phantom wallet
- Create a Candy Machine with `createCandyMachine`
- Add config lines with `addConfigLines`
- Mint NFTs with `mintFromCandyMachine`

## Stack

- React 18
- Vite
- ts-tokens
