# NFT Minting Site

A minimal React + Vite app demonstrating NFT minting with **ts-tokens**.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/stacksjs/ts-tokens/tree/main/examples/nft-minting-site)

## Setup

```bash
bun install
bun dev
```

## Features

- Connect Phantom wallet via `PhantomWalletAdapter`
- Fill in NFT name and metadata URI
- Mint an NFT using `createNFT`

## Stack

- React 18
- Vite
- ts-tokens
